# REPO Delta Force Mod v0.1.0

## 中文

### 版本概述

这是 REPO Delta Force Mod 的第一版可游玩公开版本。

这个项目的灵感来自一个“把三角洲护航摸金体验带进 REPO”的视频设想。作为同时玩三角洲和 REPO 的玩家，我想把那种特殊摸金氛围和事件感，以更适合 REPO 的方式做进游戏里，于是有了这个 mod。

我不是专业开发者，这更像是一个玩家作者边做边学完成的主题化玩法尝试。

### 本版内容

- 加入哈夫克开局事件
- 通过事件向地图投放哈夫克公司特殊物资
- 加入军用信息终端玩法
- 加入飞行记录仪玩法
- 加入航空箱玩法
- 接通特殊物资与原版 valuable / haul 统计链
- 完成公开仓库整理与源码清理

### 当前版本特点

- 已经可以正常游玩
- 核心玩法链已经打通
- 当前版本重点在于主题完整度与可玩性
- 后续是否继续长期更新，会根据玩家反馈再决定

### 后续方向

如果后续继续更新，当前比较明确的方向包括：

- 继续扩展哈夫克物资池，加入更多有趣物资
- 增加“收集一整套获得特殊奖励”之类的新玩法
- 推出阿萨拉事件
- 增加阿萨拉集团物资池
- 等事件和物资逐渐丰富后，再去调整开局事件触发概率

## English

### Overview

This is the first publicly shareable playable version of REPO Delta Force Mod.

The project was inspired by the idea of bringing a Delta Force-style escort looting experience into REPO. As a player of both Delta Force and REPO, I wanted to recreate part of that special extraction-loot atmosphere in a way that fits REPO, which is how this mod began.

I am not a professional developer. This is better described as a player-made themed gameplay project built while learning along the way.

### Included In This Release

- Havoc opening event
- Event-driven Havoc supply insertion into the map
- Military terminal gameplay
- Flight recorder gameplay
- Air drop case gameplay
- Integration with vanilla valuable / haul tracking
- Public repository cleanup and source preparation

### Current Focus

- Playable core experience
- Working end-to-end gameplay loop
- Stronger thematic identity for the first public version
- Future long-term updates will depend on player feedback

### Future Direction

If development continues, the current roadmap direction includes:

- expanding the Havoc supply pool with more interesting items
- introducing set-collection rewards or similar mechanics
- adding an Assala-themed event
- building an Assala company supply pool
- adjusting opening event trigger rates later, once the event and item pools become richer
