# REPO Delta Force Mod v0.1.0

## 中文

### 版本概述

这是 REPO Delta Force Mod 的第一版可游玩公开版本。

当前版本已经完成哈夫克主题核心玩法落地，主要围绕哈夫克开局事件和三类特殊物资展开：

- 军用信息终端
- 飞行记录仪
- 航空箱

### 本版内容

- 加入哈夫克开局事件
- 加入军用信息终端玩法
- 加入飞行记录仪玩法
- 加入航空箱玩法
- 接通特殊物资与原版 valuable / haul 统计链
- 完成正式仓库清理与公开源码整理

### 当前特点

- 已经可以正常游玩
- 核心玩法链已打通
- 当前版本重点在于可玩性与主题完整度

### 后续方向

- 继续优化投放体验与平衡
- 继续打磨 UI 与反馈表现
- 为后续主题扩展保留结构空间

### Overview

This is the first publicly shareable playable version of REPO Delta Force Mod.

The current build delivers the core Havoc-themed gameplay loop centered around the Havoc opening event and three special supplies:

- Military Terminal
- Flight Recorder
- Air Drop Case

### Included In This Release

- Havoc opening event
- Military terminal gameplay
- Flight recorder gameplay
- Air drop case gameplay
- Integration with vanilla valuable / haul tracking
- Repository cleanup for public source hosting

### Current Focus

- Playable core experience
- Working end-to-end gameplay loop
- Stronger theme identity for the first public version

### Next Steps

- Further polish spawn flow and balance
- Improve feedback and presentation
