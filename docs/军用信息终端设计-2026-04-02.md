# 军用信息终端设计（2026-04-02）

本文档用于固定第一版 `军用信息终端` 的统一设计口径。

## 1. 物资定位

军用信息终端是哈夫克物资池中的 `情报型工具物资`。

它的核心作用不是直接制造恐怖感，也不是高风险异常物件，而是：

- 帮助玩家发现高价值目标
- 提供搜集方向
- 提升玩家的情报获取效率

在哈夫克三件物资中的分工是：

- 军用信息终端负责 `找`
- 飞行记录仪负责 `险`
- 航空箱负责 `取舍`

## 2. 第一版核心玩法

军用信息终端当前第一版的玩法口径如下：

- 它被定义为 `工具`，不是必须一直手持搬运的大件 valuable
- 它可以进入 `物品栏`
- 它会以固定节奏发出 `雷达扫描`
- 扫描会提示高价值目标的大致情报

这样做的核心原因是：

- 如果它必须像普通 valuable 一样一直拿在手上，玩家会因为无法顺畅开门、交互而觉得它拖累操作
- 把它做成可放进物品栏、后台持续工作的工具，才更符合“情报终端”的定位

## 3. 当前确认的设计规则

### 3.1 扫描节奏

- 每 `7 秒` 扫描一次
- 扫描不是连续刷新的，而是有明显脉冲节奏

### 3.2 锁定规则

- 一旦锁定目标，不轻易切换
- 只有在以下情况才重新锁定：
  - 目标失效
  - 目标消失
  - 目标碎裂
  - 目标被回收换钱
  - 当前锁定不再有效

这条规则的目的，是让终端更像“稳定追踪一个情报目标”，而不是不断跳目标、让玩家难以判断。

### 3.3 HUD 规则

HUD 当前只负责给出 `情报线索`，不负责完全揭示目标。

当前确认的 HUD 输出方向：

- 显示目标价值
- 显示目标方向
- 显示模糊距离
- 显示下次扫描倒计时

当前默认不直接显示：

- 目标具体名字

这样做的目的是保留探索感，不把终端做成“完全透视答案”的工具。

补充口径（2026-04-03）：

- 军用信息终端手持时底部现有的 `6 格绿色边框 HUD` 不再视为需要强行删除的异常 UI，而是正式吸纳为终端自己的 `电量槽 / 扫描次数槽`
- 左上角扫描情报 HUD 仍然保留，底部 6 格则承担资源限制表达
- 因此军用信息终端的 HUD 现在由两部分组成：
  - 左上角：目标情报
  - 底部六格：终端电量

### 3.4 电量与目标切换规则

军用信息终端新增正式资源规则如下：

- 终端满电为 `6 格`
- `第一次锁定目标免费`
- 之后每次 `切换到下一个目标` 时，消耗 `3 格`
- 这里的“切换目标”指的是：
  - 当前目标被收集
  - 当前目标被破坏
  - 当前目标消失或失效
  - 系统因此改为指向下一个高价值目标
- 终端不是持续扫描时缓慢掉电，而是在 `目标切换瞬间` 扣电

按当前口径推算：

- 满电状态下，终端可支持 `1 次免费初始锁定 + 2 次付费切换`
- 也就是理论上单局最多连续追踪 `3 个` 高价值目标

### 3.5 电量耗尽后的表现

当军用信息终端电量耗尽后：

- 终端进入 `电量不足` 状态
- 不需要大改后台扫描系统
- 只需要在 HUD 层隐藏部分目标情报信息
- 让玩家明确知道终端已无法继续提供下一个目标的有效引导

当前默认方向是：

- 保留 HUD 基础框架
- 弱化或隐藏目标引导信息
- 用轻量表现实现“电量不足”，而不是重做整套底层逻辑

### 3.6 手持与物品栏规则

- 放在 `物品栏` 时，终端继续后台扫描
- 只有切到 `手上` 时，左上角 HUD 才显示

这意味着：

- 终端在背包里也在持续提供情报功能
- 但只有玩家主动拿出来时，才会看到完整的扫描 HUD

## 4. 当前设计边界

第一版军用信息终端的重点是 `扫描与情报`，不是负面惩罚系统。

因此以下内容当前属于 `后续补充方向`，不是第一版必须项：

- 吸引怪物
- 给玩家带来额外风险暴露
- 更复杂的声光反馈惩罚

你最早的想法里，终端的信号可能会吸引怪物，这个方向仍然成立，但当前第一版默认先不强行实装，把终端先做成稳定可用的情报工具。

## 5. 刷新与事件归属

军用信息终端属于：

- `哈夫克事件`
- `哈夫克物资池`

在物资池中的定位是：

- 普通可放置物品点即可
- 不需要像飞行记录仪那样偏角落、偏隐蔽
- 也不需要像航空箱那样优先找空旷大空间

## 6. 设计关键词

军用信息终端当前的核心关键词可以固定为：

- 工具
- 情报
- 扫描
- 锁定
- 背包可持续工作
- 手持时显示 HUD
- 六格电量槽
- 目标切换扣电

## 7. 当前默认口径

如果后续没有新的明确修改，当前线程默认按以下口径继续推进：

- 军用信息终端是情报型工具物资
- 可以进入物品栏
- 每 7 秒扫描一次
- 锁定目标后不轻易切换
- 物品栏中继续后台扫描
- 只有手持时显示 HUD
- HUD 显示价值、方向、模糊距离和倒计时
- 底部六格 HUD 正式定义为终端电量槽
- 第一次锁定目标免费
- 每次切换到下一个目标时消耗 3 格电量
- 电量耗尽后显示“电量不足”并隐藏部分情报 HUD
- 第一版暂不把“吸引怪物”作为强制实现项
