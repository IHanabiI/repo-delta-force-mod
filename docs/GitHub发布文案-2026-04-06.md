# GitHub 发布文案

## 仓库 About 描述

中文：

这是一个三角洲行动主题的 REPO 玩法模组，灵感来自“把三角洲护航摸金体验带进 REPO”的视频设想。当前第一版以哈夫克事件为起点，通过事件向地图投放军用信息终端、飞行记录仪与航空箱等特殊物资。

English:

This is a Delta Force-themed gameplay mod for REPO, inspired by the idea of bringing Delta Force-style escort looting into REPO. The current first version starts with a Havoc event prototype that injects special supplies such as the military terminal, flight recorder, and air drop case into the map.

## 未来更新方向短句

中文：

如果后续继续更新，方向不会只停留在套装奖励，而会继续扩展哈夫克物资池、加入高回报但带特殊效果或额外风险的物资、完善现有三个核心物资未完全实装的部分，并逐步推出阿萨拉事件与阿萨拉集团物资池。

English:

If development continues, the direction will go beyond set-collection rewards. It may include expanding the Havoc supply pool, introducing high-reward items with special effects or extra risk, completing the still-unfinished potential of the current three core supplies, and gradually adding an Assala event with its own company supply pool.
