# GitHub 发布文案

## 仓库 About 描述

中文：

为 REPO 制作的哈夫克主题玩法模组，灵感来自“把三角洲护航摸金体验带进 REPO”的视频设想。当前版本通过哈夫克事件向地图投放特殊物资，包括军用信息终端、飞行记录仪与航空箱。

English:

A Havoc-themed gameplay mod for REPO, inspired by the idea of bringing Delta Force-style escort looting into REPO. The current version uses a Havoc event to inject special supplies into the map, including a military terminal, a flight recorder, and an air drop case.

## 作者说明短句

中文：

这是一个玩家作者边做边学完成的项目，不是专业团队作品。后续是否继续长期更新，会根据玩家反馈再决定。

## 未来更新方向短句

中文：

如果后续继续更新，方向不只会是套装收集奖励，还会包括扩展哈夫克物资池、加入高回报但带特殊效果或额外风险的物资、继续完善现有三个核心物资的完整功能，并逐步推出阿萨拉事件与阿萨拉集团物资池。

English:

If development continues, the direction will go beyond set-collection rewards. It may include expanding the Havoc supply pool, introducing high-reward items with special effects or extra risk, continuing to complete the unfinished potential of the current three core supplies, and gradually adding an Assala event with its own company supply pool.
